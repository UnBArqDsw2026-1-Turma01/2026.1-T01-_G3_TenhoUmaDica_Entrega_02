# 2.2.2. Diagrama de Comunicação

## Descrição
O Diagrama de Comunicação (antigo Diagrama de Colaboração) é um diagrama dinâmico da UML que foca na organização dos objetos que participam de uma interação. Diferente do Diagrama de Sequência, que prioriza a ordem temporal, este diagrama enfatiza os elos de associação entre os objetos e a arquitetura de mensagens que circulam por eles.

## Objetivo
O objetivo deste artefato é demonstrar a colaboração entre as instâncias das classes (como `Membro`, `Dica`) para realizar uma funcionalidade específica do **TenhoUmaDica**. Ele permite visualizar quais objetos são mais "centrais" na arquitetura e como as responsabilidades são distribuídas entre eles.
