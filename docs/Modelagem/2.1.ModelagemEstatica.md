# 2.1. Módulo Notação UML – Modelagem Estática

## 1. Introdução
A modelagem estática descreve a estrutura lógica e física do sistema **#TenhoUmaDica**, focando nos componentes que o formam e nos relacionamentos entre as entidades de dados. Nesta etapa, não consideramos o fator tempo ou o comportamento dinâmico, mas sim como o software é construído internamente.

## 2. Objetivo
Este módulo visa fornecer uma visão clara da arquitetura do sistema, permitindo que a equipe de desenvolvimento compreenda as regras de negócio, a hierarquia de classes e a organização modular necessária para suportar as funcionalidades de gamificação e repositório de conteúdos.
